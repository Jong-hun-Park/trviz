Generate a plot for motif color map

trplot(aligned_labeled_repeats: List[str], sample_ids: List[str] = None, figure_size: Tuple[int, int] = None, output_name: str = None, dpi: int = 500, alpha: float = 0.6, box_line_width: float = 0, xtick_degrees: int = 90, hide_xticks: bool = False, hide_yticks: bool = False, symbol_to_motif: Dict = None, sort_by_clustering: bool = True, hide_dendrogram: bool = False, population_data=None, sample_as_row=False, private_motif_color: str = 'black', frame_on={'top': False, 'bottom': True, 'right': False, 'left': True}, debug: bool = False)¶

Generate a plot showing the variations in tandem repeat sequences. A distinct color is assigned to each motif. For private motifs (with low frequency), the same color (in black by default) may be assigned.

Parameters

  • aligned_labeled_repeats -- aligned and encoded tandem repeat sequences.

  • sample_ids -- sample IDs

  • figure_size -- figure size

  • output_name -- output file name

  • dpi -- DPI for the plot

  • alpha -- alpha value for the plot

  • box_line_width -- line width for box edges

  • xtick_degrees -- xtick degree (default is 90)

  • hide_xticks -- if true, hide xticks

  • hide_yticks -- if true, hide yticks

  • private_motif_color -- the color for private motifs. Default is black

  • debug -- if true, print verbse information.
